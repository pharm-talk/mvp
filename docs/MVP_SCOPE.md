# MVP 스코프

---

## 1. 포함 (MVP)

- 이메일/소셜 로그인 (Supabase Auth)
- 유저 건강정보 입력 폼 (멀티스텝 온보딩)
- 내 약 서랍
  - 약/영양제 추가 (사진 업로드 + 수동 입력)
  - 리스트 조회, 수정, 삭제
- 복약상담 요청
  - 건강정보 + 복용 리스트 + 증상 입력
  - 결제 후 상담 생성
  - 약사 답변 조회
- Q&A 질문
  - 간단 질문 입력 + 결제
  - 약사 답변/추가 질문 1회
- 약사 등록 + 면허 인증 (어드민 수동 검증)
- 약사 대시보드
  - 대기 중 상담 목록
  - 상담 상세 조회
  - 답변 작성/제출
- 결제 연동 (토스페이먼츠 또는 포트원)

---

## 2. 제외 (v2 이후)

- 실시간 채팅 상담
- 고급 OCR (혈액검사 결과지 등 복잡한 문서)
- 긴급 유료 즉시 답변 옵션
- 정기구독 커머스 (영양제 정기배송)
- 고도화된 약사 자동 매칭 알고리즘
- AI 사전 분석 → 약사 검증 하이브리드 모델
- 알림/마케팅 자동화 (리텐션 캠페인 등)
