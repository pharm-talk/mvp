# PHARMACIST FLOW (약사)

---

## 1. 가입 및 인증

1. 회원가입 (이메일/카카오 등)
2. 약사 전용 가입 플로우 진입
   - 약사 면허 번호 입력
   - 면허증 사진/스캔본 업로드
   - 추가 정보: 약국명, 근무지, 경력 등
3. 어드민이 수동으로 서류 검토 후 승인/반려
4. 승인 완료 시 약사 대시보드 접근 권한 활성화

---

## 2. 약사 대시보드

### 2.1 메인 화면

- `대기 중 상담 목록`
  - 복약상담 / Q&A 구분
  - 요청 시간, 예상 답변 마감 시간
  - 유저의 기본 정보 요약 (성별, 나이, 주요 질환 등)

### 2.2 상담 상세

1. 상담 카드 클릭 → 상세 화면 진입
2. 유저 정보 확인
   - 건강정보, 복용 리스트, 증상/질문 내용
3. 답변 작성
   - 플랫폼 제공 템플릿에 맞춰 섹션별 작성
   - (예) 현재 복용 리스트 요약, 겹치는 효능/성분, 주의 조합, 추천 조합, 추천 제품
4. 추천 제품 등록
   - 가성비 픽 / 프리미엄 픽
   - 각 제품에 대한 간단 설명, 링크 첨부
5. 답변 제출 → 상태 `answered`로 변경

---

## 3. Q&A 처리

1. Q&A 탭에서 대기 중 질문 목록 확인
2. 간단 질문에 대한 텍스트 답변 작성
3. 필요한 경우, 추가 질문을 유도하는 안내 문구 작성
4. 답변 제출 후, 유저가 1회 추가 질문 가능

---

## 4. 정산/내역

- 지난 상담 내역 리스트
- 상담별 정산 예정 금액
- 지급 완료/대기 상태
- 월별 요약 (총 상담 수, 총 정산액)
