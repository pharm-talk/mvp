# USER FLOW (일반 유저)

---

## 1. 온보딩

1. 회원가입 (이메일/카카오 등)
2. 기본 건강정보 입력
   - 성별, 나이, 키, 몸무게
   - 보유 질환, 알레르기, 임신 여부 등
3. 온보딩 완료 → 홈 진입

---

## 2. 복약상담 플로우

1. 홈 → "복약상담 받기" CTA 클릭
2. 복용 중인 약/영양제 입력
   - 내 약 서랍에서 선택
   - 또는 새로 추가 (사진 촬영 / 수동 입력)
3. 현재 증상/컨디션 입력
   - 텍스트 (필수)
   - 사진 첨부 (선택, 예: 검사 결과지)
4. 상담 요약 화면에서 최종 확인
5. 결제 화면 → 결제 완료
6. 상담 요청 생성, "대기 중" 상태
7. 약사 배정 후, 답변 작성
8. 답변 완료 시
   - 푸시/인앱 알림
   - 유저는 답변 상세 화면에서 내용 확인
   - 추천 제품 링크 클릭 가능
9. 유저는 추가 질문 1회까지 가능

---

## 3. Q&A 플로우

1. 홈 → "약사에게 질문하기" CTA 클릭
2. 질문 카테고리 선택 (선택 사항)
3. 질문 내용 텍스트 입력
4. 관련 사진 첨부 (선택, 예: 약 봉투, 피부 상태 등)
5. 결제 → 질문 등록
6. 약사 답변 대기
7. 답변 완료 알림
8. 유저는 답변 확인 후, 1회 추가 질문 가능

---

## 4. 내 약 서랍

1. 홈 → "내 약 서랍" 진입
2. 현재 복용 리스트 확인
3. 약 추가
   - 사진 촬영 → OCR 인식 → 제품 후보 리스트 → 선택
   - 또는 수동 입력 (제품명, 성분, 복용 주기 등)
4. 약 편집/삭제
5. 리필 알림 설정 (선택)

---

## 5. 마이 페이지

- 프로필/기본 정보 수정
- 건강정보 수정
- 결제 내역 및 상담 기록 리스트
- 알림 설정 (푸시, 이메일 등)
