# 💊 약사 상담 플랫폼 — PRD

---

## 1. 한 줄 요약

면허 인증 약사의 복약상담을 모바일 웹으로 제공하는 플랫폼.  
창고형 약국 시대에 사라져가는 **개인화된 복약지도**를 디지털로 되살린다.

---

## 2. 문제 정의

### 2.1 소비자 관점
- 창고형 약국에서는 복약상담 없이 약/영양제를 구매하게 됨
- 영양제를 여러 개 먹고 있지만 겹치는 성분, 상호작용을 모름
- 약국 영업시간 외(야간, 주말)에는 약사에게 물어볼 수 없음
- 인터넷 건강 정보는 신뢰도가 낮고 개인 상황에 맞지 않음

### 2.2 약사 관점
- 창고형 약국 확산으로 동네 약국 매출 감소
- 복약지도라는 전문성이 저평가되고 있음
- 약국 외 추가 수익 채널이 없음

---

## 3. 솔루션

약사 면허 인증 기반의 **비동기 1:1 복약상담 플랫폼**을 모바일 웹으로 제공한다.

### 3.1 복약 · 영양제 상담

**유저 입력 항목**
- 기본 정보: 성별, 나이, 신장(cm), 체중(kg)
- 건강 정보: 보유 질환, 과거 병력, 유전력, 알레르기, 임신 여부
- 복용 리스트: 현재 복용 중인 약/영양제 (사진 촬영 자동 인식 or 수동 입력)
- 건강검진 결과: 사진 업로드 (선택)
- 현재 증상/컨디션: 자유 텍스트 입력

**약사 답변 구성**
- 현재 복용 중인 것 중 효능 겹치는 항목 정리 → 빼도 되는 것 추천
- 같이 먹으면 효능 떨어지거나 부작용 있는 조합 경고
- 추가로 먹으면 좋은 영양제 추천 (추천 이유 포함)
- 가성비 픽 / 프리미엄 픽 두 가지 옵션 제시
- 추천 제품 → 구매 링크 연결 (쿠팡 파트너스, 아이허브 등 제휴)

### 3.2 약사 Q&A

- 유저가 증상/궁금한 점을 텍스트로 입력
- 면허 인증 약사만 답변 가능
- 비동기 1:1 전달 방식 (게시판형 X)
- 답변 수신 시 알림 (푸시 또는 인앱)
- 유저는 답변에 대해 추가 질문 1회 무료

---

## 4. 유저 타입

| 타입 | 설명 |
|------|------|
| 일반 유저 | 상담을 요청하고 답변을 받는 소비자 |
| 약사 | 면허 인증 후 상담 답변을 제공하는 전문가 |
| 어드민 | 약사 면허 검증, 신고 처리, 서비스 운영 |

---

## 5. 주요 기능

### 5.1 복약 · 영양제 상담
- 건강정보 + 복용 리스트 입력
- 상담 요청 생성 및 결제
- 약사 배정 및 답변
- 상담 결과 저장 및 재조회

### 5.2 약사 Q&A
- 간단 증상/질문 등록
- 약사 답변 수신
- 추가 질문 1회

### 5.3 사진 촬영 자동 인식 (MVP는 기본 수준)
- 영양제/약 봉투/포장지 사진 업로드
- OCR + 제품 DB 매칭으로 복용 리스트 구성
- 인식 결과 사용자 확인/수정

### 5.4 내 약 서랍
- 복용 중인 약/영양제 리스트 저장/관리
- 약 추가/삭제/수정
- 상담 기록 연결

### 5.5 약사 프로필 & 매칭
- 약사 프로필: 전문 분야, 경력, 평점, 답변 수
- 초기에는 수동/라운드로빈 매칭

### 5.6 결제
- 상담 유형별 고정 가격
- 토스페이먼츠 또는 포트원 연동
- 답변 미수신 시 환불 정책

---

## 6. 수익 모델

| 수익원 | 설명 |
|--------|------|
| 상담 건당 과금 | 유저 결제 → 플랫폼 수수료 차감 → 약사 정산 |
| 제휴 커머스 수수료 | 추천 제품 구매 시 수수료 |
| 약사 프리미엄 구독 | 상위 노출, 프로필 강조 등 (추후) |
| 정기구독 커머스 | 영양제 큐레이션 구독 박스 (추후) |

---

## 7. 차별화 포인트

- **면허 인증 약사의 실제 답변** — AI 답변이 아닌 전문가 신뢰
- **개인화된 복약지도** — 내 건강정보 기반 맞춤 상담
- **추천 → 구매 원스톱** — 상담 결과에서 바로 구매
- **시간 제약 없음** — 야간/주말에도 상담 가능
- **사진 한 장으로 시작** — 진입 장벽 최소화

## 8. MVP 스코프 요약

### 8.1 포함 (MVP)
- 이메일 + 소셜 로그인
- 유저 건강정보 입력 (멀티스텝 온보딩 폼)
- 약/영양제 사진 촬영 → OCR 인식 (기본)
- 내 약 서랍 CRUD
- 복약상담 요청 → 약사 답변 (비동기)
- Q&A 질문 → 답변 (비동기)
- 약사 등록 + 면허 인증 (수동 검증)
- 약사 대시보드 (대기 상담 목록 + 답변 작성)
- 결제 연동 (토스페이먼츠 or 포트원)
- 추천 제품 구매 링크 연결

### 8.2 제외 (v2 이후)
- 실시간 채팅 상담
- 고급 OCR (혈액검사 결과지 등)
- 긴급 유료 즉시 답변 옵션
- 정기구독 커머스
- 고도화된 자동 매칭 알고리즘
- AI 사전 분석 → 약사 검증 하이브리드 모델
